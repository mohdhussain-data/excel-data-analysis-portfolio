# Payroll Analysis (Excel)

## Overview
This project analyzes employee payroll data across 4 weeks using Microsoft Excel. It demonstrates data cleaning, calculations, pivot tables, and dashboard creation.

## Dataset
- 10 employees
- 4 weeks of payroll data
- Fields: employee_name, week, hourly_wage, hours_worked

## Key Calculations
- Base Pay = hourly_wage * hours_worked
- Overtime Hours = IF(hours_worked > 40, hours_worked - 40, 0)
- Overtime Bonus = overtime_hours * (hourly_wage * 0.5)
- Total Pay = base_pay + overtime_bonus
