# Employee Training Assessment Dashboard

## Objective
Analyze employee assessment results to evaluate overall performance, identify skill gaps, and highlight top and low performers.

---

## Dataset
- Total Employees: 22  
- Tests Conducted:
  - Safety  
  - Company Philosophy  
  - Financial Skills  
  - Drug Test  
- Each test has different maximum scores  
- Data includes employee-wise scores and calculated performance metrics  

---

## Tools Used
- Microsoft Excel  
  - Formulas (SUMIF, AVERAGEIF, COUNTIF, INDEX-MATCH)  
  - Data Cleaning  
  - Data Aggregation  
  - Charts (Bar Chart, Pie Chart)  

---

## Dashboard Overview
The dashboard provides:
- Key Performance Indicators (KPIs):
  - Total Employees  
  - Average Score  
  - Pass Rate  
  - Fail Count  
  - Top Performer  
  - Worst Performer  

- Top 5 Employees by Performance  
- Pass vs Fail Distribution  
- Average Score by Test  

---

## Key Insights
- Overall performance is low with an average score of 48%  
- 45% of employees failed at least one test  
- Financial Skills has the lowest average score (~49%), indicating a major skill gap  
- Safety test shows the highest performance (~60%), indicating better compliance understanding  
- Cara Clarke is the top performer with 93%, significantly above average  

---

## Recommendations
- Conduct focused training programs on Financial Skills  
- Provide support plans for employees scoring below 50%  
- Reassess training content for weaker areas  
- Implement periodic assessments to track improvement over time  

---

## Project Structure
- `analysis` sheet: Final dashboard and insights  
- `calculations` sheet: All derived metrics and formulas  
- `raw_data` sheet: Original dataset  

---

## Conclusion
This project demonstrates the use of Excel for transforming raw data into actionable insights through structured analysis and visualization.
